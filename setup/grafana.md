# Setup Grafana
## Download Grafana for Window:
```https://grafana.com/grafana/download```
## Grafana by Docker

```docker run -d --name=grafana -p 3000:3000 grafana/grafana-enterprise```

## Connection MySQL DB:
```
Visit: http://localhost:3000
```
